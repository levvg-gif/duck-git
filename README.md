🦆 DUCK-C (The Ultimate Quacker)
Простейшая утилита на Си, которая выводит уточку в терминал. Написана специально для тех, кому не хватало экзистенциального кряканья в их Arch Linux.
✨ Особенности
Много флагов: Спите, злитесь или умирайте вместе с уткой.
Rainbow Power: Флаг -r для тех, кто любит поярче.
Пайпы: Дружит с fortune, cowsay и grep.
Sudo-сюрприз: Попробуй запустить через sudo, если не боишься за чистоту своего ~.
🛠 Установка
Из AUR (самый правильный путь):
yay -S duck-git
Вручную (если ты хакер):
make
sudo make install
🚀 Использование:
duck -m "Привет, мир"          # Обычная утка
fortune | duck -r              # Радужная философская утка
sudo duck                      # НЕ ДЕЛАЙ ЭТОГО (или делай)
⚠ ВНИМАНИЕ (ВАЖНО!)
Эта программа поставляется под лицензией MIT.
Если вы запустите её через sudo, утка решит, что она хозяйка в доме, и совьет гнездо в вашем ~/quack. Там появится очень важный архив what.lz4.tar.gz.md5.zip.7z и скрипт для его "дешифровки".
